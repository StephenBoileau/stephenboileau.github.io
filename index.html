Live Character System – Unreal Engine

Technical Artist and Developer responsible for building a real-time live character experience for a major theme park attraction at Epic Universe. Developed systems in Unreal Engine to drive a live animated character used in daily guest interactions.

Built and maintained a robust animation pipeline with hundreds of animation states, blending systems, and performance triggers. Created a custom in-engine GUI to allow operators and performers to control the character in real time, ensuring reliable daily operation in a live entertainment environment.

Tools: Unreal Engine, C++, Blueprints, Maya, real-time animation systems, live performance tooling.
