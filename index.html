Live Character System – Unreal Engine (2024)

Technical Artist and Developer for a live Unreal Engine character experience used in a major theme park attraction.

Built and maintained a real-time animation system supporting hundreds of animation states and transitions. Developed a custom operator-friendly GUI inside Unreal Engine allowing performers and operators to control the character during live performances.

The system drives a live animated puppet performing daily for park guests and requires high reliability, fast response time, and intuitive controls for performers.

Tools: Unreal Engine, C++, Blueprints, Maya, real-time animation systems.

Moxie Robot – Real-Time Character System (2020)

Worked at Embodied Inc. as a Technical Artist helping bring the Moxie robot to market, which launched in April 2020. Moxie is a social robot designed to support child development through expressive real-time character animation and interactive experiences.

Contributed to character rigging, C# programming, Maya Python tools, asset organization and processing pipelines, and mobile graphics optimization to ensure stable real-time performance on the device.

Tools: C#, Python, Maya, Unity, mobile graphics optimization.

Chimelong Spaceship – Cute Animal Party (2022)

Joined the final stage of development on a live Unity-based character experience for Chimelong Spaceship, featuring animated characters performing in front of a live audience.

Resolved major performance issues by optimizing heavy assets in Autodesk Maya, profiling and improving Unity code, and stabilizing technical systems to address architectural bugs and evolving client needs. Increased performance from roughly 13 FPS to above 60 FPS without sacrificing visual quality.

Tools: Unity, C#, Autodesk Maya, profiling, performance optimization.

Client Theater Visualizer (2023)

Designed and built a Quest-based VR theater visualization tool that allowed clients to preview in-progress media inside accurately scaled virtual theater environments throughout development.

Created to-scale theater models in Autodesk Maya and developed precise UV mapping for complex curved projection surfaces. Built an Android/Quest application with a simple file-picker workflow so clients could load media directly onto the headset and review how content would appear on the intended screens without needing technical support.

I created several theater visualizers during my time at Super 78, though some cannot be shown publicly because the related projects have not yet been built or released.

Tools: Unity, C#, Android, Meta Quest, Autodesk Maya, UV mapping, VR visualization.

Princess Cruises – 360 Dining Experience (2023)

Worked on the Princess Cruises 360 dining experience, an immersive theatrical dining environment where projection, sound, and interactive graphics transform the dining table into part of the show.

Developed TouchDesigner and Python systems using OpenCV to track guest motion and gestures, driving real-time graphics and visual effects that respond dynamically to activity at the table.

Tools: TouchDesigner, Python, OpenCV, real-time interactive graphics.

Dino on My Desk AR (2016)

Sole developer of an educational AR app for kids called Dino on My Desk, released in 2016 for Android and iOS. The experience followed a small dinosaur collecting objects while teaching players about the character and the historical period it came from.

Built the application in Unity using C#, supported Android and iOS development, collaborated with Maya-based assets, and managed project workflows using Git.

Tools: Unity, C#, Android, iOS, Git, Autodesk Maya, AR development.
